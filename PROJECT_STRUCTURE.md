## Naming Conventions

| Artefact | Convention | Example |
|---|---|---|
| Scripts | `snake_case.py` | `feature_engineering.py` |
| Model files | `{model}_{version}.pkl` | `random_forest_v1.pkl` |
| Figures — EDA | `NN_description.png` | `05_bivariate_boxplots.png` |
| Figures — Eval | `{model}_{type}.png` | `random_forest_confusion_matrix.png` |
| Notebooks | `NN_title.ipynb` | `01_eda_exploration.ipynb` |
| Data arrays | `splits_{dim}.npz` | `splits_2d.npz` |

## Data Flow

```
data/raw/  -->  src/preprocess.py  -->  data/processed/
                       |
              models/scaler.pkl
              models/label_encoders.pkl
                       |
          src/train.py  &  src/feature_engineering.py
                       |
          models/ml/  &  models/dl/
                       |
              src/evaluate.py
                       |
        reports/figures/  &  reports/metrics/
```

## Key Config & Environment Variables

| Variable | Description | Default |
|---|---|---|
| `RANDOM_STATE` | Global seed for reproducibility | `42` |
| `TEST_SIZE` | Test split fraction | `0.20` |
| `VAL_FRACTION` | Validation fraction (from train) | `0.15` |
| `CHURN_RATE` | Target churn rate in synthetic data | `0.20` |

See `.env.example` for the full list.
